RAM - overclocking, easy, but strong plays

Thankfully, there are a couple plays anyone can do. First, Make sure you have 2 (two) identical memory sticks installed. This gives you Dual-channel. 2 sticks perform cosmically better than 1 stick. Dual-channel doubles your RAM bandwidth: You had a 2 lane road, now you have a 4 lane road. This alone gives you massive, massive performance gains, much more than anything else you might ever do for with your RAM. Depending on your configuration and task, we are talking up to 50% more performance. This is by far the best thing you can do. Second, I would recommend checking whether your specific model has XMP Profile. XMP Profile is essentially Turbo for RAM. It's a preset that's going to get you at least 50% of your max possible RAM overclock. At which point you should probably chill. I want to reiterate chasing that other half is a whole blown activity, and very autistic at that. Third, Just like you can bump/lower CPU ratio multiplier, you can as easily bump/lower RAM ratio multiplier. Real world example: I noticed that my laptop supports DDR-2933, while my Task Manager shows 2666. This is actually a very common thing for laptops. Decided to give it a lift. 1. Reboot, tap Delete to Enter BIOS 2. Go to Overclocking Performance Menu 3. Go to Memory 4. Increase the Ratio I changed Ratio from 10 to 11, which bumped my RAM speed (DDR transfer rate) from 2666 to 2933. This is nice because this was simple and free. Without going into technical detail here (and they get very technical I promise) This was actually a big overclock, at 0.1% the effort. I didn't touch voltage. I didn't touch "timings" (oh God). I directly got ~10% RAM bandwidth AND at ~9% decrease in system latency. Realistically, this gives me up to ~10% more average FPS / CPU performance in non-GPU bound scenarios, ~9% less fps drops, and also a ~9% decrease in system latency / responsiveness of everything. This is not nothing at all. So what did we just do? RAM - just like the CPU - has a core speed and a ratio multiplier. Universally, RAM core speed is 133.33 MHz. (CPU was 100 MHz if you remember) But by bumping the multiplier from 10 to 11, my RAM frequency should've gone up... ...from 1333 to 1466? What? Well, it did. That (1333/1466) frequency is actually Memory Clock - MCLK. That is, in fact, the real RAM operating frequency. Now follow the hand - DDR (as in DDR3, DDR4, DDR5) stands for...... Double Data Rate So if we multiply MCLK by Double we get... (133.33 * 10) * 2 = DDR-2666 (133.33 * 11) * 2 = DDR-2933 (133.33 * 12) * 2 = DDR-3200 Cool? Saying DDR-3200 MHz is also incorrect. Your task manager is wrong. Your BIOS is also wrong. Because they say DDR-3200 MHz. That is a lie for the sake of simplicity. Or marketing. DDR-3200 is data rate transfer speed. The correct notation for DDR is MT/s - mega transfers per second. DDR-3200 is not 3200 MHz. DDR-3200 is 3200 MT/s. DDR-3200 operates at 1600 MHz. MCLK MHz = clock speed * multiplier DDR MT/s = clock speed * multiplier * 2 So MCLK is always half DDR.
