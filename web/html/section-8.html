Basic Monitoring

You can't manage what you don't measure. You want to get a 'feel' for your machine. You want to feel its pulse. Sounds very erotic honestly, but I mean it - you need to monitor things. Go ahead and install a monitoring tool. There are a plenty of, but you're going to need MSI Afterburner. (we need it to tinker with GPU later) Set up Monitoring as shown. For our purposes we only need to see 4 things. CPU usage, CPU temperature GPU usage, GPU temperature. (if voltage control is grayed out, ignore it for now) If we want to tune our components - we have to find the 'sweet spot' on our own.
