Windows - Autolaunch

Apps like ThrottleStop, MSI Afterburner, SilentOption, and other must be running in the background to apply their effects. So let's add convenience to our day. Open WIN+R > shell:startup, and throw your app shortcuts in this folder. This is the autolaunch folder. Alternatively, you can create tasks in Task Scheduler. - Create Basic Task - When I log on / When computer starts - Start a program - Select app.
